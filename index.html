Eclipse Plugin Installation steps

  • Open Eclipse, go to help>install new software
  • In the work with text box, write http://accelamea.github.io/IDE
  • Select Accela from the list and click next to proceed with the installation

Excel Plugin Installation steps

  • Download the setup file from here
  • Make sure you have Office 2016 32 bit installed on your machine
  • Close Excel (if opened), and run the setup
